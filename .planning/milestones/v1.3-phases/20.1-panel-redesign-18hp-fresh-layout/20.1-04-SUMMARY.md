# Phase 20.1 Plan 04: WaveformDisplay px Constant Re-tune Summary

Bumped only the hardcoded-px constants inside `WaveformDisplay` so the +27% wider / +45% taller 81.44x26.00mm box (Plan 03) reads as a deliberate refit rather than a stretched artifact — proportional math and the frozen DSP/atomic/Hz-format surface left fully intact, compiles clean.
