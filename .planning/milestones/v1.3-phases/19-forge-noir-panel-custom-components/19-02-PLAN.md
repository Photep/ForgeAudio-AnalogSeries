**Typography path generation approach:**

- Brand text: 4.0mm cap height = scale factor of ~0.8 on the 5-unit-tall letterforms
- Module name: 4.0mm cap height = same ~0.8 scale factor
- Hero/Output labels: 3.20mm = scale ~0.64
- Primary labels: 1.80mm = scale ~0.36
- Small labels: 1.40mm = scale ~0.28

New letters needed beyond existing set: M, H, P, T, S (for MORPH, CHARACTER, PHASE, etc.). Construct in the same geometric block letter style: 5 units tall, proportional widths, 0.9-unit stroke weight, with chamfered/clipped corners matching existing letters. Use fill-rule="evenodd" for letters with counters (A, D, O, P, R).
