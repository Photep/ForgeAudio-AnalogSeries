User verifies the complete clock tracking infrastructure in VCV Rack with a real clock source. This is a human-verify checkpoint -- Claude has already built and installed the module. The user tests the six scenarios below.

Open VCV Rack 2. Add the Forge Audio Analog Series LFO module. Verify the following:

**1. CLK Jack Exists (CLK-01)**
- Confirm CLK input jack is visible on the panel
- Confirm you can patch a cable into it

**2. Basic Clock Tracking (CLK-02, CLK-03, CLK-06)**
- Add a clock module (e.g., VCV CLOCKED or BPM Clock)
- Set clock to ~120 BPM
- Patch clock output to CLK input
- Observe: LFO phase should reset on the first pulse (CLK-06)
- Observe: LFO should continue at its Rate knob frequency (Phase 7 does NOT override frequency)
- Note: The LFO will still run at Rate knob frequency since Phase 8 adds the frequency override. What you should see is phase resetting to 0 on each clock pulse.

**3. Clock Loss Timeout (CLK-04)**
- With clock patched and running, stop the clock source (disable/bypass the clock module)
- Observe: After a few seconds (1-5s depending on tempo), the LFO should revert to normal free-running behavior (no more phase resets)

**4. Cable Disconnect (CLK-04 related)**
- With clock patched and running, physically unplug the CLK cable
- Observe: LFO should immediately revert to free-running (no waiting for timeout)

**5. Outlier Rejection (CLK-05)**
- With clock running at 120 BPM and module tracking:
  - Quickly swap the CLK cable to a different-tempo clock source (e.g., 30 BPM or 480 BPM)
  - Observe: The LFO should NOT show wild behavior during the swap. It should either maintain the old period (outlier rejected) and then timeout and re-acquire, or smoothly transition.

**6. Re-acquisition (Fast-track)**
- With clock at 120 BPM, let it lock. Disconnect the CLK cable. Wait 2 seconds. Reconnect the same clock.
- Observe: Should re-acquire quickly (fast-track since same tempo)

Note: Since Phase 7 does NOT override frequency, the main observable behaviors are:
- Phase resets to 0 on each clock edge (visible as the phase dot jumping to the left side of the display)
- State transitions happening (no direct visual yet -- Phase 10 adds the SYNC badge)
- The LFO continues at Rate knob speed between resets
