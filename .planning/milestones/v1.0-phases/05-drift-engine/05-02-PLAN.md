  <what-built>Complete drift engine and three-knob analog engine (morph, character, drift). This is the final feature of the module -- after this verification, the module is functionally complete.</what-built>
  <how-to-verify>
    Launch VCV Rack and add the Forge Audio Analog LFO module.

    **1. Drift knob basic behavior:**
    - Set Morph to sine (fully CCW), Character to 0, Rate to ~1Hz
    - Verify with Drift at zero: phase dot moves at perfectly steady speed
    - Slowly turn Drift up: observe phase dot begin to speed up and slow down irregularly
    - At full Drift: phase dot should have clearly visible speed variation (multi-timescale -- some slow wandering, some faster jitter)

    **2. Drift does NOT change waveform shape:**
    - With Drift at full: verify the waveform TRACE stays the same shape (only the dot speed changes)
    - Move Morph through all shapes while Drift is active -- shapes should be correct

    **3. Drift CV input:**
    - Set Drift knob to 0, connect an LFO or other CV source to the Drift CV input
    - Turn up the Drift attenuator trimpot (the new one between CCV and DCV jacks)
    - Verify drift amount responds to the CV input

    **4. Per-instance independence:**
    - Add a SECOND Analog LFO module with identical settings
    - Both should be drifting but with DIFFERENT patterns (not in sync)

    **5. Three-knob engine integration:**
    - Set Morph to ~0.6 (saw-ish), Character to ~0.7 (strong analog), Drift to ~0.5
    - Verify the module produces a characterful, gently wandering LFO output
    - Connect the output to something audible (e.g., a VCO FM input or filter cutoff)

    **6. Panel layout:**
    - Verify the bottom row has 7 components in grouped pairs: [trimpot jack] [trimpot jack] [trimpot jack] [jack]
    - All labels are readable and centered under their respective components
    - No components overlap or look cramped
    - Overall panel aesthetics look correct

    **7. Subtle dot instability visual:**
    - At high drift, observe if the phase dot trail shows subtle jitter/wobble
    - This should be barely noticeable at mid-drift, more visible at full drift
    - Should NOT look broken or erratic
  </how-to-verify>
