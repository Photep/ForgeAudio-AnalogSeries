  <how-to-verify>
The module should already be built and installed from Plan 03-01.

1. **Launch VCV Rack** and add the AnalogLFO module
2. **Check overall display aesthetic:**
   - Does the dark navy background look like an embedded screen?
   - Is the amber glow effect subtle and modern (not retro)?
   - Does the inset frame create visual depth?
3. **Check waveform trace:**
   - At morph=0 (sine): clean, smooth amber trace?
   - Sweep morph slowly: does every intermediate shape render smoothly?
   - At morph=1 (square): clean corners, no artifacts?
4. **Check phase dot:**
   - Is the dot visible and precise (not too large)?
   - Can you see the comet trail behind it?
   - Is the dot center brighter than the trace (white-amber vs amber)?
   - Does the glow halo around the dot look good?
5. **Check slow rate behavior:**
   - Set rate to minimum (0.01Hz) -- does the dot show a gentle breathe/pulse?
6. **Check bypass behavior:**
   - Right-click the module and select Bypass -- does the display dim?
7. **Check display size:**
   - Does the display feel "generous" enough? (Currently ~19.5% of panel height, user specified 25-35%)
   - If too small, we can expand it in a follow-up adjustment
8. **Check room brightness:**
   - Reduce VCV Rack room brightness (View > Room brightness) -- display should stay visible

**Specific things to look for:**
- Glow too bright or too dim?
- Dot too large or too small?
- Trail too long or too short?
- Colors need adjustment?
- Display needs to be taller?

Type "approved" if the display looks good, or describe any adjustments needed.
