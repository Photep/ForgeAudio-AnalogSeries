<task type="checkpoint:human-verify" gate="blocking">
  <name>Task 1: Verify character knob in VCV Rack</name>
  <what-built>Complete character knob implementation with four per-shape analog reference models (Minimoog saw, SH-101/Juno square, Moog/Prophet triangle, triangle-derived sine), Character CV input with attenuator, progressive knob curve, and display integration.</what-built>
  <action>Build, install, and present verification steps to user for visual/functional confirmation in VCV Rack.</action>
  <how-to-verify>
    1. Build and install: `make && make install`
    2. Open VCV Rack, add the Forge Audio Analog LFO module
    3. **Panel layout check:** Verify bottom row has 6 evenly spaced components (MATrim, MCV, CATrim, CCV, DCV, OUT) with readable labels

    4. **Character at zero (digital perfection):**
       - Set Character knob to 0, sweep Morph through all shapes
       - Display should show clean, sharp digital waveforms (identical to before this phase)

    5. **Saw character (Minimoog Model D):**
       - Set Morph to ~60% (saw region)
       - Slowly sweep Character from 0 to 100%
       - Display should show: slight exponential curvature developing in the ramp, soft rounded reset at the top
       - The effect should be subtle and warm, not exaggerated

    6. **Square character (Roland SH-101/Juno-106):**
       - Set Morph to ~85% (square region)
       - Sweep Character 0 to 100%
       - Display should show: edges softening from sharp digital to slightly rounded, very slight duty cycle asymmetry
       - Should still clearly read as a square wave at full character

    7. **Triangle character (Moog/Prophet):**
       - Set Morph to ~35% (triangle region)
       - Sweep Character 0 to 100%
       - Display should show: sharp peaks rounding off, very slight slope asymmetry
       - Peaks should be visibly smoother at full character vs zero

    8. **Sine character (triangle-derived):**
       - Set Morph to 0% (sine region)
       - Sweep Character 0 to 100%
       - Display should show: very subtle thickening/distortion of the sine
       - Effect is the most subtle of all shapes -- should be barely visible but present

    9. **Progressive curve feel:**
       - On any shape, slowly sweep Character from 0 to 100%
       - The first half of travel should produce subtle coloring
       - The second half should produce noticeably stronger deformations
       - Most of the knob range should produce musically interesting variations

    10. **Character + Morph interaction:**
        - Set Character to 100%, sweep Morph through full range
        - Transitions between shapes should be smooth on the display
        - No jarring artifacts at morph transition points

    11. **Character CV test:**
        - Add a second LFO module, patch its output to Character CV input
        - Turn Character CV attenuator up
        - Display should show the waveform shape changing in time with the modulation LFO
        - With Character knob at 0 and attenuator at 100%, CV should sweep full character range

    12. **Zero-cost bypass:**
        - With Character at 0 and no CV patched, the module should behave identically to pre-phase behavior
  </how-to-verify>
