**2. Add display buffer members** inside AnalogLFO struct, after `double phase = 0.0;`:
```cpp
// Display double buffer (lock-free audio-to-GUI transfer)
static constexpr int DISPLAY_SAMPLES = 256;
std::array<float, DISPLAY_SAMPLES> displayBuffers[2] = {};
std::atomic<int> displayReadIdx{0};
std::atomic<float> displayPhase{0.f};

// Display update tracking
float prevDisplayMorph = -1.f;
double prevPhaseForDisplay = 0.0;
```

**3. Add updateDisplayBuffer() method** after `computeMorphedWave()`:
```cpp
void updateDisplayBuffer(float morph) {
    int writeIdx = 1 - displayReadIdx.load(std::memory_order_relaxed);
    for (int i = 0; i < DISPLAY_SAMPLES; i++) {
        float p = (float)i / (float)DISPLAY_SAMPLES;
        displayBuffers[writeIdx][i] = computeMorphedWave(p, morph);
    }
    displayReadIdx.store(writeIdx, std::memory_order_release);
}
```

**4. Pre-compute initial display buffer** at end of constructor (after `configOutput` call):
```cpp
updateDisplayBuffer(0.f);
```

**5. Add display update logic in process()**, after the morph clamp line (`float morph = rack::math::clamp(...)`) and before `float p = (float)phase;`:
```cpp
// Update display phase
displayPhase.store((float)phase, std::memory_order_relaxed);

// Update display buffer on phase wrap or morph change
bool phaseWrapped = (phase < prevPhaseForDisplay);
bool morphChanged = (std::fabs(morph - prevDisplayMorph) > 0.002f);
if (phaseWrapped || morphChanged) {
    updateDisplayBuffer(morph);
    prevDisplayMorph = morph;
}
prevPhaseForDisplay = phase;
```

**WaveformDisplay struct** (place between AnalogLFO and AnalogLFOWidget):

The struct extends `rack::widget::TransparentWidget`. It stores a raw pointer to `AnalogLFO* module` (set by the widget constructor), animation state for the breathe effect, and previous-frame phase for movement detection.

**Constants and colors:**
- Horizontal margin inside display: 4.f pixels
- Vertical margin inside display: 6.f pixels
- Trace line weight: 1.5f (core pass)
- Dot radius: proportional to display height, approximately `box.size.y * 0.03f` (about 2.5px at typical display sizes)
- Trail: 4 dots, spacing 0.015 phase units
- Amber trace: `nvgRGBAf(0.91f, 0.66f, 0.22f, alpha)` (#e8a838)
- Dark navy background: `nvgRGBAf(0.051f, 0.051f, 0.102f, 1.f)` (#0d0d1a)
- Dot center (white-amber): `nvgRGBAf(1.f, 0.91f, 0.63f, alpha)` (#ffe8a0)

**step() override:**
Advance breathe animation phase at ~0.8Hz. Use a fixed 60fps assumption for the increment (breathe is decorative, exact timing does not matter):
```cpp
breathePhase += 2.f * M_PI * 0.8f / 60.f;
if (breathePhase > 2.f * M_PI) breathePhase -= 2.f * M_PI;
```
Call `TransparentWidget::step()`.

**drawLayer(const DrawArgs& args, int layer) override:**
All rendering on layer 1 (self-illuminating). Structure:
```cpp
if (layer == 1) {
    NVGcontext* vg = args.vg;
    nvgSave(vg);
    nvgScissor(vg, 0, 0, box.size.x, box.size.y);

    drawBackground(vg);
    drawInsetFrame(vg);

    if (module) {
        int readIdx = module->displayReadIdx.load(std::memory_order_acquire);
        const auto& buffer = module->displayBuffers[readIdx];
        float phase = module->displayPhase.load(std::memory_order_relaxed);
        float dimFactor = module->isBypassed() ? 0.25f : 1.f;

        drawWaveformTrace(vg, buffer, dimFactor);
        drawPhaseDot(vg, buffer, phase, dimFactor);
    } else {
        drawPlaceholder(vg);
    }

    nvgResetScissor(vg);
    nvgRestore(vg);
}
TransparentWidget::drawLayer(args, layer);
```

**drawBackground(NVGcontext* vg):**
Fill a rounded rect (corner radius 2.0f) with dark navy color, covering the full widget bounds.

**drawInsetFrame(NVGcontext* vg):**
Two strokes for the embedded-screen look:
1. Outer shadow: dark border at `nvgRGBAf(0, 0, 0, 0.3)`, width 1.0, at full widget bounds, rounded corners.
2. Inner highlight: faint amber at `nvgRGBAf(0.91, 0.66, 0.22, 0.15)`, width 0.5, inset 0.5px, rounded corners.

**drawWaveformTrace(NVGcontext* vg, buffer, dimFactor):**
Four-pass glow rendering. For each pass:
```
Pass 0: width=6.0, alpha=0.04*dimFactor
Pass 1: width=4.0, alpha=0.08*dimFactor
Pass 2: width=2.5, alpha=0.15*dimFactor
Pass 3: width=1.5, alpha=0.85*dimFactor (core trace)
```
For each pass, build a polyline path from the 256-sample buffer:
- `nvgBeginPath(vg);`
- For i=0: `nvgMoveTo(vg, phaseToX(0), valueToY(buffer[0]));`
- For i=1..255: `nvgLineTo(vg, phaseToX(i/256.f), valueToY(buffer[i]));`
- Set `nvgLineCap(vg, NVG_ROUND)` and `nvgLineJoin(vg, NVG_ROUND)`.
- Set stroke color to amber with per-pass alpha.
- Set stroke width.
- `nvgStroke(vg);`

**drawPhaseDot(NVGcontext* vg, buffer, phase, dimFactor):**

1. **Movement detection and breathe:**
   Compute movement since last frame. Handle wrap-around:
   ```cpp
   float movement = std::fabs(phase - prevFramePhase);
   if (movement > 0.5f) movement = 1.f - movement;  // wrapped
   float breatheFactor = 1.f;
   if (movement < 0.001f) {
       breatheFactor = 0.8f + 0.2f * std::sin(breathePhase);
   }
   prevFramePhase = phase;
   ```

2. **Comet trail** (draw before dot so dot renders on top):
   4 trail dots, spacing 0.015 phase units behind current position:
   ```
   for i = 3 down to 0:
       trailPhase = phase - (i+1)*0.015
       if trailPhase < 0: trailPhase += 1.0
       tx = phaseToX(trailPhase)
       ty = valueToY(interpolateBuffer(buffer, trailPhase))
       trailAlpha = 0.3 * (1.0 - (i+1)/5.0) * dimFactor * breatheFactor
       trailRadius = dotRadius * (0.5 + 0.5 * (1.0 - (i+1)/5.0))
       draw filled circle at (tx, ty) with amber color at trailAlpha
   ```

3. **Glow halo:**
   ```cpp
   float x = phaseToX(phase);
   float y = valueToY(interpolateBuffer(buffer, phase));
   float haloRadius = dotRadius * 3.f;
   NVGpaint halo = nvgRadialGradient(vg, x, y, 0.f, haloRadius,
       nvgRGBAf(1.f, 0.9f, 0.5f, 0.3f * dimFactor * breatheFactor),
       nvgRGBAf(1.f, 0.9f, 0.5f, 0.f));
   nvgBeginPath(vg);
   nvgCircle(vg, x, y, haloRadius);
   nvgFillPaint(vg, halo);
   nvgFill(vg);
   ```

4. **Bright center dot:**

   ```cpp
   nvgBeginPath(vg);
   nvgCircle(vg, x, y, dotRadius);
   nvgFillColor(vg, nvgRGBAf(1.f, 0.91f, 0.63f, 1.f * dimFactor * breatheFactor));
   nvgFill(vg);
   ```

**Coordinate helpers:**
```cpp
float phaseToX(float phase) const {
    float margin = 4.f;
    return margin + phase * (box.size.x - 2.f * margin);
}
float valueToY(float value) const {
    float margin = 6.f;
    return box.size.y / 2.f - value * (box.size.y / 2.f - margin);
}
float interpolateBuffer(const std::array<float, 256>& buffer, float phase) const {
    float idx = phase * 256.f;
    int i0 = (int)idx;
    int i1 = (i0 + 1);
    if (i0 >= 256) i0 = 255;
    if (i1 >= 256) i1 = 0;  // wrap for interpolation at boundary
    if (i0 < 0) i0 = 0;
    float frac = idx - (float)(int)idx;
    return buffer[i0] + frac * (buffer[i1] - buffer[i0]);
}
```
