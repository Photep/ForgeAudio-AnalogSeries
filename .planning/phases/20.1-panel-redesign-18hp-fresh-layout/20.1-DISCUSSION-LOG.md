## Knob Art vs Widgets

The fresh panel SVG draws full metal knob graphics at every anchor, while the Forge knob widgets are SvgKnobs that already render their own static body (`_bg.svg`) + rotating indicator — two knob bodies competing.

| Option | Description | Selected |
|--------|-------------|----------|
| Widget covers art | Treat SVG metal circles as mockup; widget renders on top and hides them (match/exceed radii). Reuses components unchanged. Risk: stray ring if widget smaller than bezel. | |
| Strip art, widget owns knob | Remove metal knob graphics from fresh.svg (keep soft shadow/anchor); widget owns the entire knob — same proven pattern as current 14HP panel. No double-render risk. | ✓ |
| Art as bezel ring | Keep SVG metal as recessed bezel; size widget cap smaller so panel ring frames it. Most design work — tune radii per anchor. | |

**User's choice:** Strip art, widget owns knob
**Notes:** Matches the established 14HP pattern where the panel SVG drew no knob bezels. Avoids double knob bodies.
