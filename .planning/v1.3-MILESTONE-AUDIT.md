1. **DSP→GUI atomic publish** — `displayClockEdge` (Phase 21) added beside existing display atomics, single-writer/single-reader, no race introduced on Phase 18's `process()`.

5. **plugin.json slug** — `ForgeAudio-AnalogSeries` / `ForgeAnalogLFO` consistent across plugin.json, createModel, plugin.hpp/cpp.

**E2E flow (clock in → lock → PWM-morphed waveform on 18HP panel → per-edge SYNC flash): PASS.**
