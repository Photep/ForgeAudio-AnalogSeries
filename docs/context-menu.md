# Context Menu

Right-click the module for the **Swing** submenu — the only context-menu item. Swing is active only in clocked mode; free-running mode stores 0.5 (no warp). The selection persists as `swingIndex` in the patch JSON.

| Index | Option | Fraction |
|-------|--------|----------|
| 0 | Straight 50% | 0.50 (default) |
| 1 | Light 54% | 0.54 |
| 2 | Medium 58% | 0.58 |
| 3 | Triplet 66% | 0.66 |
| 4 | Heavy 71% | 0.71 |
| 5 | Max 75% | 0.75 |
