# Install

## VCV Library

Add **ForgeAnalogLFO** from the VCV Library:

1. Subscribe to the plugin in the [VCV Library](https://library.vcvrack.com/).
2. Open VCV Rack and sync your library (Library → Update all).
3. The module appears under **Forge Audio** in the module browser.

## Manual install

Install the `.vcvplugin` artifact directly:

1. Download the `.vcvplugin` file.
2. Place it in the Rack user plugins folder (`Documents/Rack2/plugins-<os>/`).
3. Relaunch Rack.

Module not appearing? Fully quit and relaunch Rack, then re-check the plugin list.
